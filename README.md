该软件使用 MIT 开源协议
该软件由 C++编写 编译器 JetBrain CLion
若你已经安装 g++或者是 C++环境 您可以使用该仓库中的 Chat_run.bat 对它进行一个编译

**如何使用？**
Step1. 安装 MinGW 环境
Step2. 打开 CMD 尝试输入 g++ 如果有回应那 MinGW 就安装成功了
Step3. 运行 Chat_run.bat 进行构建
Step4. 文件夹会出现一个chat.exe 双击运行
Step5. 该软件使用 WebUI 代理在http://127.0.0.1:8080 即可使用该软件
